    Grocy deployment notes

    The deployment has been really easy thanks to the Linuxserver docker. I've used the simple-deploy-docker ansible role to create a service based on docker.

    The configuration is straightforward, most is done through the UI, but you can disable the Chores, Tasks, and other stuff if you don't use it in the config.php file.
